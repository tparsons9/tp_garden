The perspective of the value of the [[Information Structure|signal]] $\sigma$ relates the signal to the [[Signal-induced Belief Distribution]]. Given an uninformative signal, the agent's posterior is _deterministically equal_ to the agents [[Prior Probability|prior]], so there is no dispersion. 

If the signal reveals the state $\theta$ directly, the [[Posterior Probability|posterior belief]] is a point mass on the true state, which _"maximally varies"_ depending on the realization of the signal. 

The core goal of characterizing information structures is comparing the "informativeness", see [[Instrumental Value of the Signal]]. We may expect more informative signals to be associated with more _dispersed beliefs_, but the **measure** of the dispersion is important. For example, using variance to measure dispersion yields a complete order on signals, which is not equivalent to the partial Blackwell order. 

Put simply, given a **valid** measure, dispersion in beliefs yields more intuitive understanding of the informativeness of a signal. A signal being more _informative_ [[Feasible Action Distributions|allows the agent to more effectively tailor his action to the state]]. 

Measures of **pure dispersion** like [[Convex Order]] allows the conclusion that more dispersed beliefs are yielded from more informative signals, another is [[Mean-Preserving Spread]]; but [[Second-Order Stochastically Dominant]] partial orders do not. To see why this is, see [[Relationship between Convex Order and SOSD]]. 

The [[Blackwell Order|Blackwell Equivalence]] relates the measures that yield intuitive comparisons of [[Information Structure|information structures]].
